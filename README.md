MQ7 is a gas sensor module used to detect carbon monoxide (CO) levels in the air. It utilizes a tin dioxide semiconductor sensor to measure CO concentration, often integrated with a heater for optimal performance. This module is commonly employed in gas leak detection systems and carbon monoxide alarms.

Blynk is an IoT (Internet of Things) platform that empowers users to develop mobile applications capable of remotely controlling hardware devices. It offers a user-friendly interface for building customized mobile apps, along with libraries and APIs to seamlessly integrate hardware components and sensors with cloud services. Blynk supports various microcontrollers such as Arduino, Raspberry Pi, ESP8266, and ESP32, making it a popular choice for both hobbyists and professionals to prototype and deploy IoT projects.
